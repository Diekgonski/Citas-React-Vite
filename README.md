# Citas con ReactJS y Vite

Deployment del proyecto de citas de veterinaría en el cual se implementa un CRUD básico usando Hooks y props de react, así mismo se utiliza el LocalStorage para que los datos que se escriban tengan persistencia.

## Instalación

Ejecutar el comando:

```
npm install
```

_Esperar a que instale todas las dependecias del proyecto_
